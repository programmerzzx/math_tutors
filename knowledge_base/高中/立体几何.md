## 第二章 空间直线与平面

### 2.1 平面的基本性质

**公理一**：如果一条直线上的两个点在一个平面内，那么这条直线上所有的点都在这个平面内。

**公理二**：经过不在同一条直线上的三点，有且只有一个平面。

**公理三**：如果两个不重合的平面有一个公共点，那么它们有且只有一条通过该点的公共直线。

### 2.2 空间直线的位置关系

| 关系 | 定义 | 判定 |
|------|------|------|
| 相交 | 有一个公共点 | — |
| 平行 | 没有公共点，且在同一平面内 | 同位角相等 |
| 异面 | 没有公共点，且不在同一平面内 | — |

**异面直线的判定**：平面外一点与平面内一点的连线与平面内不过该点的直线是异面直线。

### 2.3 直线与平面的位置关系

| 位置关系 | 公共点个数 | 记号 |
|----------|------------|------|
| 直线在平面内 | 无数个 | l ⊂ α |
| 直线与平面相交 | 一个 | l ∩ α = A |
| 直线与平面平行 | 零个 | l ∥ α |

## 第三章 直线、平面平行的判定与性质

### 3.1 直线与平面平行

**判定定理**：
如果平面外一条直线与平面内的一条直线平行，那么该直线与平面平行。

```
l ⊄ α
m ⊂ α
l ∥ m
⇒ l ∥ α
```

**性质定理**：
如果一条直线与一个平面平行，经过该直线的平面与该平面相交，那么该直线与交线平行。

```
l ∥ α
l ⊂ β
α ∩ β = m
⇒ l ∥ m
```

### 3.2 平面与平面平行

**判定定理**：
如果一个平面内的两条相交直线与另一个平面平行，那么这两个平面平行。

```
a ⊂ α, b ⊂ α
a ∩ b = A
a ∥ β, b ∥ β
⇒ α ∥ β
```

**性质定理**：
两平行平面被第三个平面所截，截线平行。

## 第四章 直线、平面垂直的判定与性质

### 4.1 直线与平面垂直

**定义**：如果一条直线垂直于平面内的任意一条直线，则该直线垂直于这个平面。

**判定定理**：
如果一条直线垂直于平面内的两条相交直线，那么该直线垂直于这个平面。

```
l ⊥ m, l ⊥ n
m ⊂ α, n ⊂ α
m ∩ n = A
⇒ l ⊥ α
```

**性质定理**：
垂直于同一平面的两条直线平行。

### 4.2 平面与平面垂直

**定义**：两平面相交，如果所成的二面角是直二面角，则这两个平面垂直。

**判定定理**：
如果一个平面过另一个平面的垂线，那么这两个平面垂直。

```
l ⊥ α
l ⊂ β
⇒ α ⊥ β
```

**性质定理**：
如果两平面垂直，则一个平面内垂直于交线的直线垂直于另一个平面。

## 第五章 空间向量

### 5.1 空间直角坐标系

**右手系**：拇指指向x轴正方向，食指指向y轴正方向，中指指向z轴正方向。

**坐标表示**：
```
点 A(x, y, z)
向量 a⃗ = (a₁, a₂, a₃)
```

### 5.2 空间向量运算

**加法**：a⃗ + b⃗ = (a₁+b₁, a₂+b₂, a₃+b₃)

**数乘**：λa⃗ = (λa₁, λa₂, λa₃)

**数量积**：a⃗ · b⃗ = a₁b₁ + a₂b₂ + a₃b₃ = |a||b|cosθ

**向量积**：a⃗ × b⃗ = (a₂b₃ - a₃b₂, a₃b₁ - a₁b₃, a₁b₂ - a₂b₁)

### 5.3 法向量

**平面的法向量**：垂直于平面的向量。

**求法**：
1. 设 n⃗ = (x, y, z)
2. 在平面内找两个不共线向量 a⃗, b⃗
3. 解方程 n⃗ · a⃗ = 0, n⃗ · b⃗ = 0

### 5.4 空间向量应用

#### 证明平行
```
l ∥ α ⟺ l的方向向量与α的法向量垂直
l ⊥ α ⟺ l的方向向量与α的法向量平行
```

#### 证明垂直
```
l ⊥ α ⟺ l的方向向量平行于α的法向量
```

#### 求二面角
```
设二面角两个面的法向量为 n⃗₁, n⃗₂
cosθ = |n⃗₁ · n⃗₂| / (|n⃗₁||n⃗₂|)
```

#### 求点到平面距离
```
d = |n⃗ · PA| / |n⃗|
其中 P 为平面外一点，A 为平面上任意一点
```

## 第六章 证明技巧

### 6.1 传统法证明步骤

1. **审题**：明确已知条件和要求证的结论
2. **画图**：准确画出空间图形
3. **分析**：找出关键的几何关系
4. **表达**：用数学语言规范书写

### 6.2 向量法证明步骤

1. **建系**：建立适当的空间直角坐标系
2. **求点**：求出相关点的坐标
3. **求向量**：表示出相关向量
4. **计算**：利用向量运算解决问题
5. **结论**：得出几何结论

### 6.3 常见模型

#### 翻折问题
- 翻折前后，不变的面和线段保持不变
- 变化的量：面与面所成的二面角

#### 截面问题
- 找截面与各棱的交点
- 连线要构成平面

#### 最值问题
- 转化为函数最值
- 常用方法：向量法、几何法
