# 大学数学证明 - 抽象代数

## 第一章 群论基础

### 1.1 群的定义

**定义**：设G是一个非空集合，在G上定义一个二元运算·，满足：
1. **结合律**：∀a,b,c∈G，(a·b)·c = a·(b·c)
2. **单位元存在**：∃e∈G，使得 ∀a∈G，e·a = a·e = a
3. **逆元存在**：∀a∈G，∃a⁻¹∈G，使得 a·a⁻¹ = a⁻¹·a = e

则称(G, ·)为一个群。

**有限群与无限群**：群的阶 |G| 是有限还是无限。

**Abel群（交换群）**：额外满足交换律 a·b = b·a。

### 1.2 证明是群

**例题**：证明 (Z, +) 是群。

**证明**：
```
1. 结合律：∀a,b,c∈Z，(a+b)+c = a+(b+c) ✓
2. 单位元：0∈Z，使得 a+0 = 0+a = a ✓
3. 逆元：∀a∈Z，-a∈Z，使得 a+(-a) = (-a)+a = 0 ✓

∴ (Z, +) 是群
```

### 1.3 子群

**定义**：设H是群G的非空子集，若H对G的运算也构成群，则称H为G的子群，记作 H ≤ G。

**子群判定定理**：
H ≤ G ⟺ 
1. H非空
2. ∀a,b∈H，ab⁻¹∈H

**或者**：
H ≤ G ⟺
1. H非空
2. ∀a,b∈H，ab∈H
3. ∀a∈H，a⁻¹∈H

### 1.4 陪集与指数

**左陪集**：aH = {ah | h∈H}

**性质**：
- aH = bH ⟺ a⁻¹b ∈ H
- |aH| = |H|
- 陪集划分G

**指数**：G/H 的基数，记作 [G:H]

**Lagrange定理**：若H ≤ G，则 |G| = |H| · [G:H]

## 第二章 同态与同构

### 2.1 同态映射

**定义**：设φ: G → G' 是映射，若 ∀a,b∈G，
```
φ(ab) = φ(a)φ(b)
```
则称φ为G到G'的同态映射。

**核**：Ker φ = {g∈G | φ(g) = e'}

**像**：Im φ = {φ(g) | g∈G}

### 2.2 同构映射

**定义**：若同态φ还是双射，则称φ为同构映射。

**同构记号**：G ≅ G'

### 2.3 同态基本定理

**定理**：若 φ: G → G' 是满同态，则
```
G/Ker φ ≅ G'
```

**证明思路**：
1. 定义自然映射 η: G → G/Ker φ
2. 证明 η 是满同态，Ker η = Ker φ
3. 构造 φ̄: G/Ker φ → G'，使得 φ = φ̄ ∘ η
4. 证明 φ̄ 是同构

## 第三章 循环群与置换群

### 3.1 循环群

**定义**：若存在 a∈G，使得 G = {aⁿ | n∈Z}，则称G为由a生成的循环群，记作 ⟨a⟩。

**例题**：证明 (Zₙ, +ₙ) 是循环群。

**证明**：
```
取 a = 1
则 aⁿ = n mod n = 0,1,2,...,n-1

∴ ⟨1⟩ = Zₙ
∴ (Zₙ, +ₙ) 是由1生成的循环群
```

### 3.2 置换群

**n次对称群**：Sₙ = {所有n元置换}

**奇置换与偶置换**：σ可分解为k个对换，k为奇/偶决定奇/偶。

**Aₙ（交错群）**：Sₙ中所有偶置换构成的子群，|Aₙ| = n!/2

## 第四章 正规子群与商群

### 4.1 正规子群

**定义**：若H ≤ G，且 ∀g∈G，gHg⁻¹ = H，则称H为G的正规子群，记作 H ⊲ G。

**等价条件**：
- H ⊲ G ⟺ ∀g∈G，gHg⁻¹ ⊆ H
- H ⊲ G ⟺ ∀g∈G，∀h∈H，ghg⁻¹ ∈ H
- H ⊲ G ⟺ ∀g∈G，gH = Hg

### 4.2 商群

**定义**：设 H ⊲ G，定义商群 G/H = {gH | g∈G}，运算为
```
(g₁H)(g₂H) = (g₁g₂)H
```

**例题**：证明 (Z, +) / (nZ) ≅ (Zₙ, +ₙ)

**证明**：
```
∵ (nZ) ⊲ (Z, +)（Abel群的子群必正规）

定义 φ: Z → Zₙ，φ(m) = m mod n

φ 是满同态，Ker φ = nZ

由同态基本定理：
Z/Ker φ ≅ Zₙ
即 Z/nZ ≅ Zₙ
```

## 第五章 环论基础

### 5.1 环的定义

**定义**：设R是一个非空集合，定义加法和乘法，满足：
1. (R, +) 是Abel群
2. 乘法结合律
3. 乘法对加法的分配律：a(b+c) = ab + ac, (a+b)c = ac + bc

则称(R, +, ·)为环。

**交换环**：乘法满足交换律

**含单位元环**：存在1，使得 1·a = a·1 = a

### 5.2 域

**定义**：若环R同时满足：
1. (R\{0}, ·) 是Abel群
2. 乘法对加法的分配律

则称R为域。

**例题**：证明 (Q, +, ·) 是域。

**证明**：
```
1. (Q, +) 是Abel群 ✓
2. (Q\{0}, ·) 是Abel群
   - 乘法封闭：a/b · c/d = ac/bd ∈ Q
   - 结合律、交换律 ✓
   - 单位元：1
   - 逆元：a/b 的逆为 b/a
3. 分配律 ✓

∴ (Q, +, ·) 是域
```

## 第六章 理想与商环

### 6.1 理想

**定义**：设I是环R的非空子集，若：
1. I对加法封闭
2. ∀r∈R，∀a∈I，ra∈I 且 ar∈I

则称I为R的理想。

**主理想**：由单个元素生成的理想
```
(a) = {ra + na | r∈R, n∈Z}
```

### 6.2 商环

**定义**：设I ⊲ R，定义商环
```
R/I = {r + I | r∈R}
运算：(r₁+I) + (r₂+I) = (r₁+r₂) + I
     (r₁+I)(r₂+I) = (r₁r₂) + I
```

## 第七章 群作用与Sylow定理

### 7.1 群作用

**定义**：设G是群，X是一个集合，若映射 G × X → X 满足：
1. e·x = x
2. (g₁g₂)·x = g₁·(g₂·x)

则称G作用在X上。

### 7.2 Sylow定理

**第一Sylow定理**：
若 |G| = pᵐm，(p,m)=1，则G的p-子群阶数为pᵐ。

**第二Sylow定理**：
所有Sylow p-子群共轭。

**第三Sylow定理**：
Sylow p-子群的个数 nₚ 满足：
1. nₚ ≡ 1 (mod p)
2. nₚ | m
