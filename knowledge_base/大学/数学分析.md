# 大学数学证明 - 数学分析

## 第一章 实数与极限

### 1.1 实数基本定理

**确界原理**：非空有上界的实数集必有上确界；非空有下界的实数集必有下确界。

**Dedekind切割定理**：若把实数集分成两个非空集合A、B，使得A∪B=R，且∀a∈A, ∀b∈B有a<b，则要么A有最大数，要么B有最小数。

### 1.2 数列极限的ε-N定义

**定义**：
```
lim(n→∞) aₙ = a

⟺ ∀ε>0, ∃N∈N*, 使得当 n>N 时，恒有 |aₙ - a| < ε
```

**证明数列极限的步骤**：
1. 写出目标不等式 |aₙ - a| < ε
2. 对不等式进行变形，解出 n > f(ε)
3. 取 N = ⌈f(ε)⌉
4. 验证：当 n > N 时，|aₙ - a| < ε

**例题**：证明 lim(n→∞) (2n+1)/(3n+2) = 2/3

**证明**：
```
∀ε>0，要使 |(2n+1)/(3n+2) - 2/3| < ε

|(2n+1)/(3n+2) - 2/3| = |(6n+3-6n-4)/(9n+6)|
                      = 1/(9n+6)
                      < 1/(9n)

要使 1/(9n) < ε，即 n > 1/(9ε)

取 N = ⌈1/(9ε)⌉

当 n > N 时，
|(2n+1)/(3n+2) - 2/3| < 1/(9n) < ε

∴ lim(n→∞) (2n+1)/(3n+2) = 2/3
```

## 第二章 函数极限

### 2.1 函数极限的ε-δ定义

**x→x₀时的极限**：
```
lim(x→x₀) f(x) = A

⟺ ∀ε>0, ∃δ>0，使得当 0<|x-x₀|<δ 时，恒有 |f(x)-A|<ε
```

**证明函数极限的步骤**：
1. 写出目标不等式 |f(x) - A| < ε
2. 分析 |x - x₀| 的范围
3. 通过不等式放缩，给出 δ = g(ε)
4. 验证

**例题**：证明 lim(x→1) (x² - 1)/(x - 1) = 2

**证明**：
```
∀ε>0，要使 |(x²-1)/(x-1) - 2| < ε

当 x ≠ 1 时，(x²-1)/(x-1) = x+1

|(x²-1)/(x-1) - 2| = |x+1-2| = |x-1|

要使 |x-1| < ε

取 δ = ε

当 0<|x-1|<δ 时，
|(x²-1)/(x-1) - 2| = |x-1| < ε

∴ lim(x→1) (x²-1)/(x-1) = 2
```

### 2.2 单侧极限

**右极限**：
```
lim(x→x₀⁺) f(x) = A ⟺ ∀ε>0, ∃δ>0, 当 0 < x - x₀ < δ 时，|f(x)-A| < ε
```

**左极限**：类似定义

## 第三章 连续性

### 3.1 连续的定义

**点连续**：
```
f在x₀连续 ⟺ lim(x→x₀) f(x) = f(x₀)
           ⟺ ∀ε>0, ∃δ>0, 当|x-x₀|<δ时，|f(x)-f(x₀)|<ε
```

**区间连续**：f在区间I上每一点都连续。

### 3.2 一致连续

**定义**：
```
f在区间I上一致连续 ⟺ ∀ε>0, ∃δ>0，使得对任意x₁,x₂∈I，
                   只要|x₁-x₂|<δ，就有|f(x₁)-f(x₂)|<ε
```

**注意**：一致连续要求 δ 只依赖于 ε，而与 x 的位置无关。

**例题**：证明 f(x) = x² 在 [0,1] 上一致连续，但在 [0,+∞) 上不一致连续。

**证明**：
```
在 [0,1] 上：
∀ε>0，要使 |x₁² - x₂²| = |x₁-x₂||x₁+x₂| < ε

当 x₁, x₂ ∈ [0,1] 时，|x₁+x₂| ≤ 2

只需 |x₁-x₂| < ε/2

取 δ = ε/2，则当 |x₁-x₂|<δ 时，|x₁²-x₂²| < ε

∴ f(x) = x² 在 [0,1] 上一致连续

在 [0,+∞) 上：
取 ε = 1
∀δ>0，取 x₁ = n, x₂ = n + δ/2（n 充分大）
则 |x₁-x₂| = δ/2 < δ
但 |x₁²-x₂²| = |n² - (n+δ/2)²| 
            = | -nδ - δ²/4 | → +∞ (当 n→∞)

∴ f(x) = x² 在 [0,+∞) 上不一致连续
```

## 第四章 微分学

### 4.1 导数定义

```
f'(x₀) = lim(Δx→0) [f(x₀+Δx) - f(x₀)]/Δx
```

### 4.2 微分中值定理

#### Rolle定理
若 f ∈ C[a,b] 且 f 在 (a,b) 内可导，且 f(a) = f(b)，则 ∃ξ ∈ (a,b)，使得 f'(ξ) = 0。

**证明思路**：
1. 由最值定理，f在[a,b]上取得最大值M和最小值m
2. 若M=m，则f为常数，结论显然
3. 若M≠m，则最值在内部取得，导数为0

#### Lagrange中值定理
若 f ∈ C[a,b] 且 f 在 (a,b) 内可导，则 ∃ξ ∈ (a,b)，使得
```
f'(ξ) = [f(b) - f(a)]/(b - a)
```

#### Cauchy中值定理
若 f,g ∈ C[a,b]，f,g 在 (a,b) 内可导，且 g'(x) ≠ 0，则 ∃ξ ∈ (a,b)，使得
```
f'(ξ)/g'(ξ) = [f(b) - f(a)]/[g(b) - g(a)]
```

### 4.3 Taylor公式

**带Peano余项**：
```
f(x) = f(x₀) + f'(x₀)(x-x₀) + ... + f⁽ⁿ⁾(x₀)/n! · (x-x₀)ⁿ + o((x-x₀)ⁿ)
```

**带Lagrange余项**：
```
f(x) = f(x₀) + f'(x₀)(x-x₀) + ... + f⁽ⁿ⁾(ξ)/n! · (x-x₀)ⁿ, ξ在x₀与x之间
```

## 第五章 积分学

### 5.1 定积分定义

**Darboux和**：
- 大和：上确界
- 小和：下确界

**可积条件**：
- f在[a,b]上有界
- f的不连续点集是零测集

### 5.2 微积分基本定理

**Newton-Leibniz公式**：
```
∫ₐᵇ f(x)dx = F(b) - F(a)，其中 F' = f
```

### 5.3 广义积分

**无穷积分**：
```
∫ₐ⁺∞ f(x)dx 收敛 ⟺ ∀ε>0, ∃M>a，使得当 b₁,b₂>M 时，
                              |∫ᵦ₁ᵦ₂ f(x)dx| < ε
```

**瑕积分**：类似定义

## 第六章 常数项级数

### 6.1 级数收敛定义

```
∑uₙ 收敛 ⟺ 部分和数列 Sₙ 有极限
```

### 6.2 正项级数判别法

| 判别法 | 条件 | 结论 |
|--------|------|------|
| 比较判别法 | uₙ ≤ cvₙ，∑vₙ收敛 | ∑uₙ收敛 |
| 比值判别法 | lim uₙ₊₁/uₙ = ρ | ρ<1收敛，ρ>1发散 |
| 根值判别法 | lim uₙ^(1/n) = ρ | 同上 |
| 积分判别法 | f(n)=uₙ，单调递减 | ∑f(n)与∫f(x)dx同敛散 |

### 6.3 任意项级数

**绝对收敛**：∑|uₙ| 收敛

**条件收敛**：∑uₙ 收敛但 ∑|uₙ| 发散

**Leibniz判别法**：若 uₙ ≥ uₙ₊₁ ≥ 0 且 lim uₙ = 0，则 ∑(-1)ⁿ⁻¹uₙ 收敛。

## 第七章 函数项级数

### 7.1 一致收敛

**定义**：
```
∑uₙ(x) 在区间I上一致收敛 ⟺ ∀ε>0, ∃N，当 n>N 时，
                           对所有 x∈I，|Rₙ(x)| = |∑ₖ₌ₙ₊₁∞ uₖ(x)| < ε
```

### 7.2 一致收敛判别法

**Weierstrass M判别法**：
若 ∑Mₙ 收敛，且 |uₙ(x)| ≤ Mₙ 对所有 x∈I 成立，则 ∑uₙ(x) 一致收敛。

### 7.3 和函数的分析性质

若 ∑uₙ(x) 在区间I上一致收敛，且每项连续，则和函数连续、可积，且可逐项积分。

若 ∑uₙ(x) 在区间I上一致收敛，且每项有连续导数，则和函数可导，且可逐项求导。
