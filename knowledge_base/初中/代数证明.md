# 初中数学证明 - 代数证明

## 第一章 实数与代数式

### 1.1 实数的基本性质

**运算律**：
- 加法交换律：a + b = b + a
- 加法结合律：(a + b) + c = a + (b + c)
- 乘法交换律：ab = ba
- 乘法结合律：(ab)c = a(bc)
- 分配律：a(b + c) = ab + ac

### 1.2 乘法公式

| 公式 | 表达式 |
|------|--------|
| 平方差 | (a + b)(a - b) = a² - b² |
| 完全平方 | (a ± b)² = a² ± 2ab + b² |
| 立方和 | (a + b)(a² - ab + b²) = a³ + b³ |
| 立方差 | (a - b)(a² + ab + b²) = a³ - b³ |

### 1.3 因式分解

**常用方法**：
1. 提取公因式法
2. 运用公式法
3. 十字相乘法
4. 分组分解法
5. 换元法

**步骤**：
1. 提 → 先考虑提取公因式
2. 套 → 检查能否运用公式
3. 分 → 考虑分组分解
4. 查 → 检查是否分解彻底

## 第二章 方程与方程组

### 2.1 一元二次方程

**一般形式**：ax² + bx + c = 0 (a ≠ 0)

**求根公式**：
```
x = (-b ± √(b² - 4ac)) / 2a
```

**根的判别式**：Δ = b² - 4ac
- Δ > 0：两个不相等的实数根
- Δ = 0：两个相等的实数根
- Δ < 0：无实数根

**韦达定理**：
```
x₁ + x₂ = -b/a
x₁ · x₂ = c/a
```

### 2.2 证明方程有解/无解

**证明有解的常用方法**：
1. 计算判别式 Δ ≥ 0
2. 构造完全平方
3. 利用函数图像

**证明无解的常用方法**：
1. 计算判别式 Δ < 0
2. 反证法：假设有解，导出矛盾

### 2.3 证明根的关系

**例题**：证明方程 x² - (m+2)x + 2m + 1 = 0 的两个根都是正数。

**证明**：
```
设方程两根为 x₁, x₂

（1）Δ = (m+2)² - 4(2m+1)
     = m² + 4m + 4 - 8m - 4
     = m² - 4m
     = (m-2)² ≥ 0
     ∴ 方程有实数根

（2）x₁ + x₂ = m + 2
     x₁ · x₂ = 2m + 1
     
     由 x₁ + x₂ > 0，得 m > -2
     由 x₁ · x₂ > 0，得 m > -1/2
     
     ∴ 当 m > -1/2 时，两根均为正数
```

## 第三章 不等式

### 3.1 不等式的基本性质

| 性质 | 表达式 |
|------|--------|
| 对称性 | a > b ⟺ b < a |
| 传递性 | a > b, b > c ⟹ a > c |
| 加法保号 | a > b ⟹ a + c > b + c |
| 乘法保号 | a > b, c > 0 ⟹ ac > bc |
| 倒数 | a > b > 0 ⟹ 1/a < 1/b |

### 3.2 证明不等式

**常用方法**：

#### 比较法
```
证明：a² + b² ≥ 2ab

a² + b² - 2ab = (a - b)² ≥ 0
∴ a² + b² ≥ 2ab
```

#### 综合法
从已知条件出发，利用不等式的性质逐步推演。

#### 分析法
从结论出发，逆向分析直到与已知条件吻合。

#### 反证法
假设结论不成立，导出矛盾。

### 3.3 一元二次不等式

**解题步骤**：
1. 将不等式化为标准形式
2. 求对应方程的根
3. 画抛物线草图
4. 根据图像写解集

## 第四章 函数

### 4.1 一次函数

**一般形式**：y = kx + b (k ≠ 0)

**与坐标轴的交点**：
- x轴交点：(-b/k, 0)
- y轴交点：(0, b)

**与不等式的关系**：
- y > kx + b ⟺ 函数图像在x轴上方
- y < kx + b ⟺ 函数图像在x轴下方

### 4.2 二次函数

**一般形式**：y = ax² + bx + c (a ≠ 0)

**顶点坐标**：
```
(-b/2a, (4ac-b²)/4a)
```

**对称轴**：x = -b/2a

**开口方向**：
- a > 0：开口向上
- a < 0：开口向下

### 4.3 函数图像的应用

**证明函数恒正/恒负**：
1. 配方成完全平方
2. 判断a的符号
3. 结合Δ的符号

## 第五章 证明技巧总结

### 5.1 代数证明的常见策略

1. **化简**：将复杂式子化简
2. **变形**：等价变形保持不变
3. **换元**：引入新变量简化问题
4. **分类讨论**：处理含参数的情况
5. **数形结合**：利用几何意义

### 5.2 证明格式规范

```
证明：
左边 = ...
     = ...
     = ...（步骤）
右边 = ...

∴ 左边 = 右边

原命题成立。
```

### 5.3 常见错误

1. 跳步推理
2. 忽略条件（如a ≠ 0）
3. 符号错误
4. 分类不完整
