# 高中数学证明 - 数列与数学归纳法

## 第一章 数列

### 1.1 等差数列

**定义**：从第2项起，每一项与它的前一项的差等于同一个常数。

**通项公式**：
```
an = a1 + (n-1)d
```

**前n项和公式**：
```
Sn = n(a1 + an)/2 = n·a1 + n(n-1)d/2
```

**性质**：
- 若 m + n = p + q，则 am + an = ap + aq
- 等差中项：A = (a + b)/2

### 1.2 等比数列

**定义**：从第2项起，每一项与它的前一项的比等于同一个常数。

**通项公式**：
```
an = a1 · q^(n-1)
```

**前n项和公式**：
```
当 q ≠ 1：Sn = a1(1 - q^n)/(1 - q)
当 q = 1：Sn = n·a1
```

**性质**：
- 若 m + n = p + q，则 am · an = ap · aq
- 等比中项：G = ±√(ab)

## 第二章 数学归纳法

### 2.1 数学归纳法原理

数学归纳法是基于自然数的良序性质：

**第一步（归纳基础）**：证明当 n = n₀（通常为1）时命题成立。

**第二步（归纳假设）**：假设当 n = k（k ≥ n₀）时命题成立。

**第三步（归纳证明）**：利用归纳假设，证明当 n = k + 1 时命题成立。

### 2.2 数学归纳法证明步骤

**例题**：用数学归纳法证明：
```
1 + 3 + 5 + ... + (2n-1) = n²
```

**证明**：

**第一步**：当 n = 1 时，
```
左边 = 1
右边 = 1² = 1
左边 = 右边
∴ 当 n = 1 时，等式成立
```

**第二步**：假设当 n = k 时，等式成立，即
```
1 + 3 + 5 + ... + (2k-1) = k²
```

**第三步**：当 n = k + 1 时，
```
左边 = 1 + 3 + 5 + ... + (2k-1) + (2k+1)
     = k² + (2k+1)     （由归纳假设）
     = k² + 2k + 1
     = (k+1)²
右边 = (k+1)²
左边 = 右边
∴ 当 n = k + 1 时，等式成立
```

**综上所述**，对任意 n ∈ N*，等式成立。

### 2.3 归纳法的变式

#### 强化归纳法
当直接证明 n = k ⇒ n = k+1 困难时，可证明更强的命题。

#### 第二数学归纳法
**步骤**：
1. 验证 n = 1, 2 时命题成立
2. 假设 n ≤ k 时命题成立
3. 证明 n = k + 1 时命题成立

#### 反向归纳法
从某个较大的值开始，向下归纳。

### 2.4 常见错误

1. **忘记验证首项**
2. **归纳假设使用不当**
3. **从 n = k+1 推到 n = k**（方向错误）
4. **跳步**：跳过关键推导步骤

## 第三章 数列求和

### 3.1 常用求和方法

#### 错位相减法
适用于 {an·bn} 型，其中 {an} 等差，{bn} 等比。

#### 裂项相消法
将通项分裂成两项之差。

#### 分组求和法
将数列分组，分别求和。

#### 倒序求和法
利用对称性。

### 3.2 裂项技巧

```
1/[n(n+1)] = 1/n - 1/(n+1)
1/[(2n-1)(2n+1)] = 1/2 · [1/(2n-1) - 1/(2n+1)]
1/[√(n+1) + √n] = √(n+1) - √n
```

## 第四章 递推数列

### 4.1 递推关系

**一阶线性递推**：
```
an+1 = pan + q
→ 构造等比数列
→ an = (a1 + q/(1-p)) · p^(n-1) - q/(1-p)
```

**二阶线性递推**：
```
an+2 = p·an+1 + q·an
→ 特征方程 r² = pr + q
→ 求根，分情况讨论
```

### 4.2 证明递推关系

**例题**：已知 a₁ = 1，a₂ = 3，an+2 = 3an+1 - 2an。
求证：an = 2^n - 1

**证明**（使用第二数学归纳法）：
```
第一步：n = 1 时，a₁ = 1，2¹ - 1 = 1，成立
       n = 2 时，a₂ = 3，2² - 1 = 3，成立

第二步：假设 n ≤ k 时成立
       即 ak = 2^k - 1，ak+1 = 2^(k+1) - 1

第三步：ak+2 = 3ak+1 - 2ak
       = 3(2^(k+1) - 1) - 2(2^k - 1)
       = 3·2^(k+1) - 3 - 2^(k+1) + 2
       = 2·2^(k+1) - 1
       = 2^(k+2) - 1

∴ 原命题成立
```
