# 大学数学证明 - 拓扑学

## 第一章 拓扑空间

### 1.1 度量空间回顾

**度量**：设X是集合，d: X×X → R，若满足：
1. 正定性：d(x,y) ≥ 0，d(x,y) = 0 ⟺ x = y
2. 对称性：d(x,y) = d(y,x)
3. 三角不等式：d(x,z) ≤ d(x,y) + d(y,z)

则称d为X上的度量，(X,d)为度量空间。

### 1.2 拓扑定义

**拓扑空间**：设X是集合，τ ⊆ P(X)，满足：
1. ∅ ∈ τ, X ∈ τ
2. 任意有限个τ中元素的交仍在τ中
3. 任意个τ中元素的并仍在τ中

则称τ为X上的拓扑，(X,τ)为拓扑空间。

τ中的元素称为开集。

### 1.3 常见的拓扑

**离散拓扑**：τ = P(X)

**平凡拓扑**：τ = {∅, X}

**余有限拓扑**：τ = {∅} ∪ {U ⊆ X | X\U 有限}

**余可数拓扑**：τ = {∅} ∪ {U ⊆ X | X\U 可数}

## 第二章 邻域、内点与闭包

### 2.1 邻域

**邻域定义**：设x∈X，N ⊆ X，若存在开集U使得 x∈U ⊆ N，则称N为x的邻域。

**邻域系**：x的所有邻域构成的族，记作N(x)。

### 2.2 内点与内部

**内点**：设A ⊆ X，x∈A，若存在x的邻域N使得 N ⊆ A，则称x为A的内点。

**内部**：A的所有内点构成的集合，记作A°或int(A)。

**性质**：
- A° 是包含在A中的最大开集
- A° = ∪{U ⊆ A | U是开集}
- A° ⊆ A

### 2.3 闭包

**闭包定义**：设A ⊆ X，包含A的最小闭集称为A的闭包，记作Ā或cl(A)。

**闭包性质**：
- Ā 是包含A的最小闭集
- Ā = ∩{F ⊇ A | F是闭集}
- A ⊆ Ā
- Ā = X \ (X \ A)°

**闭集**：F是闭集 ⟺ F̄ = F

### 2.4 边界

**边界**：∂A = Ā \ A°

**性质**：
- ∂A = Ā ∩ (X \ A)°
- ∂A = ∂(X \ A)
- A是闭集 ⟺ ∂A ⊆ A
- A是开集 ⟺ ∂A ⊆ X \ A

## 第三章 连续映射

### 3.1 连续的定义

**度量空间中连续**：
```
f: (X,d) → (Y,ρ) 在x₀连续 ⟺ 
∀ε>0, ∃δ>0，使得 d(x,x₀)<δ ⟹ ρ(f(x),f(x₀))<ε
```

**拓扑空间中连续**：
```
f: X → Y 连续 ⟺ 
对Y中任意开集V，f⁻¹(V)在X中也是开集
```

**等价定义**：
- f连续 ⟺ 对Y中任意闭集F，f⁻¹(F)在X中也是闭集
- f连续 ⟺ ∀x∈X，∀f(x)的邻域N，f⁻¹(N)是x的邻域

### 3.2 连续映射的例子

**恒等映射**：id_X: X → X 连续

**包含映射**：若A ⊆ X，i: A → X 连续

**常值映射**：f(x) = c 连续

### 3.3 连续映射的性质

**复合**：连续映射的复合仍连续

**局部性质**：若f在x₀连续，则f在x₀附近有连续性

## 第四章 紧致性

### 4.1 紧致性的定义

**紧致**：拓扑空间X是紧致的 ⟺ X的任意开覆盖有有限子覆盖。

**列紧**（在度量空间中）：X是列紧的 ⟺ X中任意序列有收敛子列。

**可分紧**：X是可分紧的 ⟺ X中任意无限子集有极限点。

### 4.2 Heine-Borel定理

**Rⁿ中的紧集**：
```
K ⊆ Rⁿ 紧 ⟺ K是有界闭集
```

**证明思路**：
1. 有界 ⟹ 存在有界开区间覆盖
2. 闭 ⟹ 开覆盖有子覆盖
3. 反之，有界闭集被开区间覆盖 ⟹ 有限子覆盖

### 4.3 紧空间的性质

**闭集是紧的**：紧空间的闭子集是紧的

**紧集是闭的**：紧集在Hausdorff空间中必是闭集

**紧映射**：f将紧集映射为紧集

**连续映射保持紧性**：紧空间的连续像是紧的

**紧空间上的连续函数**：连续函数在紧集上必能取得最大最小值

## 第五章 连通性

### 5.1 连通的定义

**连通**：X连通 ⟺ X不能表示为两个非空不相交开集之并。

**等价定义**：
- X连通 ⟺ X只有∅和X既是开集又是闭集
- X不连通 ⟺ ∃非空开集U,V，使得 U∩V=∅, U∪V=X

### 5.2 连通性的例子

**实数区间**：R中区间（无论开闭）都是连通的

**离散空间**：若|X|>1，则不连通

**拓扑学家的正弦曲线**：连通

### 5.3 道路连通

**道路**：连续映射 γ: [0,1] → X

**道路连通**：∀x,y∈X，∃道路 γ 使得 γ(0)=x, γ(1)=y

**性质**：
- 道路连通 ⟹ 连通
- 连通 ⟹/⇐ 道路连通

### 5.4 连通分支

**连通分支**：X的极大连通子集

**性质**：
- 连通分支是闭集
- 连通分支的内部可能是空的

## 第六章 分离公理

### 6.1 T₀空间

∀x≠y，存在开集包含其中一个但不包含另一个。

### 6.2 T₁空间

∀x≠y，存在开集包含x但不包含y。

**等价于**：单点集是闭集。

### 6.3 T₂空间（Hausdorff空间）

∀x≠y，存在不相交开集U,V使得 x∈U, y∈V。

**性质**：Hausdorff空间中收敛序列的极限唯一。

### 6.4 T₃空间

正则 + T₁

**正则**：∀闭集F和点x∉F，∃不相交开集U,V使得 x∈U, F⊆V

### 6.5 T₄空间

正规 + T₁

**正规**：∀不相交闭集F,G，∃不相交开集U,V使得 F⊆U, G⊆V

**重要性**：正规空间上连续函数可延拓（Urysohn引理）

## 第七章 积空间与商空间

### 7.1 积拓扑

**有限积**：X×Y 的基为 {U×V | U∈τ_X, V∈τ_Y}

**无限积**（Tychonoff积）：积拓扑的基为投影的基的开集之有限交

### 7.2 商拓扑

**商映射**：π: X → X/~ 为自然投影

**商拓扑**：X/~ 的拓扑使得 π 连续 ⟺ U 在 X/~ 中开 ⟺ π⁻¹(U) 在 X 中开

### 7.3 商空间的例子

**圆周**：S¹ = [0,1] / (0~1)

**环面**：T² = S¹ × S¹

**Möbius带**：[0,1]×[0,1] / (0,y)~(1,1-y)

---

## 第八章 同伦与基本群

### 8.1 同伦

**同伦**：连续映射 f,g: X → Y，若存在连续映射 H: X×[0,1] → Y 使得
```
H(x,0) = f(x)
H(x,1) = g(x)
```
则称f与g同伦，记作 f ≃ g。

### 8.2 收缩与不动点

**收缩核**：A ⊆ X，若存在收缩映射 r: X → A 使得 r|_A = id_A

**Brouwer不动点定理**（二维）：
闭单位盘 D² 到自身的连续映射必有不动点。

### 8.3 基本群

**闭路**：γ: [0,1] → X, γ(0)=γ(1)=x₀

**道路类**：同伦的闭路视为同一类

**基本群**：π₁(X,x₀)，道路类的乘法构成群

**性质**：
- π₁(X×Y) ≅ π₁(X) × π₁(Y)
- 圆周 S¹ 的基本群同构于 Z
