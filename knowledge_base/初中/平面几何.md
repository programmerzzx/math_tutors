# 初中数学证明 - 平面几何

## 第一章 三角形

### 1.1 三角形的基本性质

**定义**：由不在同一条直线上的三条线段首尾顺次连结所组成的图形叫做三角形。

**分类**：
- 按边分类：等边三角形、等腰三角形、不等边三角形
- 按角分类：锐角三角形、直角三角形、钝角三角形

### 1.2 三角形的重要线段

| 线段 | 定义 | 性质 |
|------|------|------|
| 中线 | 顶点与对边中点的连线 | 三条中线交于一点（重心） |
| 高 | 顶点到对边的垂线 | 三条高交于一点（垂心） |
| 角平分线 | 顶点与对边成等角 | 三条角平分线交于一点（内心） |

### 1.3 三角形全等的判定

#### 边边边（SSS）
**定理**：三边分别对应相等的两个三角形全等。

**格式**：
```
在△ABC和△DEF中
AB = DE（已知）
BC = EF（已知）
AC = DF（已知）
∴ △ABC ≅ △DEF（SSS）
```

#### 边角边（SAS）
**定理**：两边及其夹角分别对应相等的两个三角形全等。

**注意**：必须是夹角，不是任意角。

#### 角边角（ASA）
**定理**：两角及其夹边分别对应相等的两个三角形全等。

#### 角角边（AAS）
**定理**：两角及其中一角的对边分别对应相等的两个三角形全等。

#### 斜边直角边（HL）
**定理**：斜边和一条直角边分别对应相等的两个直角三角形全等。

**注意**：只能用于直角三角形。

### 1.4 证明全等的常用技巧

1. **寻找公共边/角**
   - 图形中共享的边或角往往相等
   
2. **利用中点**
   - 中点可构造等量关系
   
3. **利用垂直**
   - 垂线段相等
   
4. **利用平行**
   - 同位角相等、内错角相等

### 1.5 常见全等证明模型

#### 手拉手模型
两个有公共顶点的三角形，绕顶点旋转后全等。

#### 8字模型
两个三角形共用一条边，形成"8"字形。

#### 一线三等角模型
一条直线上有三个相等的角，常配合全等使用。

## 第二章 平行线

### 2.1 平行线的判定

#### 同位角相等，两直线平行
```
如果 ∠1 = ∠2
那么 AB ∥ CD
```

#### 内错角相等，两直线平行
```
如果 ∠1 = ∠2
那么 AB ∥ CD
```

#### 同旁内角互补，两直线平行
```
如果 ∠1 + ∠2 = 180°
那么 AB ∥ CD
```

### 2.2 平行线的性质

#### 两直线平行，同位角相等
```
如果 AB ∥ CD
那么 ∠1 = ∠2
```

#### 两直线平行，内错角相等
```
如果 AB ∥ CD
那么 ∠1 = ∠2
```

#### 两直线平行，同旁内角互补
```
如果 AB ∥ CD
那么 ∠1 + ∠2 = 180°
```

### 2.3 证明平行的常用方法

1. **同位角相等 → 平行**
2. **内错角相等 → 平行**
3. **同旁内角互补 → 平行**
4. **平行公理的推论**：平行于同一直线的两直线平行
5. **三角形中位线**：中位线平行于第三边

## 第三章 特殊四边形

### 3.1 平行四边形

**定义**：两组对边分别平行的四边形。

**判定方法**：
1. 两组对边分别平行
2. 两组对边分别相等
3. 一组对边平行且相等
4. 对角线互相平分

**性质**：
- 对边平行且相等
- 对角相等
- 对角线互相平分

### 3.2 矩形

**定义**：有一个角是直角的平行四边形。

**判定**：有三个角是直角的四边形

**性质**：
- 具有平行四边形的所有性质
- 四个角都是直角
- 对角线相等

### 3.3 菱形

**定义**：一组邻边相等的平行四边形。

**判定**：
- 四条边都相等
- 对角线互相垂直的平行四边形

**性质**：
- 具有平行四边形的所有性质
- 四条边相等
- 对角线互相垂直
- 每条对角线平分一组对角

### 3.4 正方形

**定义**：一组邻边相等且有一个角是直角的平行四边形。

**性质**：
- 四条边相等
- 四个角都是直角
- 对角线相等且互相垂直平分

## 第四章 圆

### 4.1 圆的基本概念

**定义**：到定点的距离等于定长的点的集合。

**基本元素**：圆心、半径、直径、弦、弧。

### 4.2 圆的相关证明

#### 垂径定理
垂直于弦的直径平分这条弦，并且平分弦所对的两条弧。

**逆定理**：平分弦（不是直径）的直径垂直于弦。

#### 圆周角定理
**定理**：同弧所对的圆周角等于它所对的圆心角的一半。

**推论**：同弧或等弧所对的圆周角相等。

#### 圆内接四边形
**定理**：圆内接四边形的对角互补。

**逆定理**：对角互补的四边形内接于圆。

### 4.3 切线的证明

**判定方法**：
1. 经过半径的外端且垂直于半径
2. 圆心到直线的距离等于半径

**性质**：
- 切线垂直于经过切点的半径
- 经过圆心且垂直于切线的直线必经过切点

## 第五章 证明格式规范

### 5.1 几何语言规范

```
证明：∵ ... （条件）
     ... （推理）
     ∴ ... （结论）
     
即：∵ 公共边AB = AB
    ∠ABC = ∠DEF（已知）
    BC = EF（已知）
    ∴ △ABC ≅ △DEF（SAS）
```

### 5.2 常见书写错误

1. ∵∴位置颠倒
2. 理由不充分
3. 跳步太多
4. 对应关系错误

### 5.3 证明思路

1. **审题**：看清题目要求，明确已知和求证
2. **画图**：准确画出几何图形
3. **分析**：找出关键条件和隐藏条件
4. **表达**：规范书写证明过程
5. **检验**：检查证明的完整性和正确性
