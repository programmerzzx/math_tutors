# 高中数学证明 - 不等式证明

## 第一章 不等式的基本性质

### 1.1 基本性质

| 性质 | 表达式 |
|------|--------|
| 对称性 | a > b ⟺ b < a |
| 传递性 | a > b, b > c ⟹ a > c |
| 加法保序 | a > b ⟹ a + c > b + c |
| 乘法保序（正数） | a > b, c > 0 ⟹ ac > bc |
| 乘法反序（负数） | a > b, c < 0 ⟹ ac < bc |
| 倒数（正数） | a > b > 0 ⟹ 1/a < 1/b |
| 乘方（正数） | a > b > 0 ⟹ aⁿ > bⁿ |

### 1.2 常用不等式

**均值不等式**：
```
a² + b² ≥ 2ab
a + b ≥ 2√(ab) （a, b > 0）
```

**基本不等式链**：
```
调和平均 ≤ 几何平均 ≤ 算术平均 ≤ 平方平均
H ≤ G ≤ A ≤ Q
```

## 第二章 证明方法

### 2.1 比较法

**作差比较**：
```
证明 a > b
⟺ 证明 a - b > 0
⟺ 变形为易于判断的完全平方等形式
```

**例题**：证明 x² + 3 > 3x

**证明**：
```
x² + 3 - 3x = x² - 3x + 3
            = (x - 3/2)² + 3 - 9/4
            = (x - 3/2)² + 3/4 > 0

∴ x² + 3 > 3x
```

### 2.2 综合法

从已知条件或已证不等式出发，利用不等式的性质逐步推演。

**例题**：已知 a, b > 0，a + b = 1
求证：√(a + 1/2) + √(b + 1/2) ≤ 2

**证明**：
```
∵ a, b > 0，a + b = 1
∴ 0 < a < 1，0 < b < 1

令 x = √(a + 1/2)，y = √(b + 1/2)
则 x² + y² = a + b + 1 = 2

(x + y)² = x² + y² + 2xy = 2 + 2xy ≤ 2 + x² + y² = 4

∴ x + y ≤ 2

即 √(a + 1/2) + √(b + 1/2) ≤ 2
```

### 2.3 分析法

从待证结论出发，逐步逆推，直到与已知条件吻合。

**格式**：
```
要证 A > B
只需证 ...
只需证 ...
...
显然成立
∴ 原不等式成立
```

**例题**：证明 √(a² + b²) ≥ (a + b)/2

**证明**：
```
要证 √(a² + b²) ≥ (a + b)/2
只需证 4(a² + b²) ≥ (a + b)²
只需证 4a² + 4b² ≥ a² + 2ab + b²
只需证 3a² - 2ab + 3b² ≥ 0
只需证 (a - b)² + 2(a² + b²) ≥ 0

显然成立
∴ 原不等式成立
```

### 2.4 反证法

假设结论不成立，导出矛盾。

**适用情形**：
- 结论以否定形式出现
- 结论是唯一的情况
- 结论涉及"至少"、"至多"

**例题**：证明 √2 是无理数

**证明**：
```
假设 √2 是有理数
则 √2 = p/q（p, q互质）
平方得 2 = p²/q²
即 p² = 2q²

∴ p² 是偶数，∴ p 是偶数
设 p = 2k

则 4k² = 2q² ⟹ q² = 2k²
∴ q² 是偶数，∴ q 是偶数

∴ p, q 都是偶数，与 p, q 互质矛盾

∴ 假设不成立
∴ √2 是无理数
```

### 2.5 放缩法

有意识地将不等式一边放大或缩小，使证明简化。

**常用放缩**：
```
1/k² < 1/(k-1) - 1/k
1/k(k+1) = 1/k - 1/(k+1)
2√(k+1) - 2√k = 2/(√(k+1) + √k) < 1/√k
```

**例题**：证明 1 + 1/2² + 1/3² + ... + 1/n² < 2

**证明**：
```
1/k² < 1/(k-1) - 1/k（当 k ≥ 2）

∴ 1 + 1/2² + 1/3² + ... + 1/n²
< 1 + (1 - 1/2) + (1/2 - 1/3) + ... + (1/(n-1) - 1/n)
= 1 + 1 - 1/n
< 2
```

## 第三章 重要不等式

### 3.1 柯西不等式

**二维形式**：
```
(a² + b²)(c² + d²) ≥ (ac + bd)²
当 ad = bc 时取等
```

**证明**：
```
(a² + b²)(c² + d²) - (ac + bd)²
= a²c² + a²d² + b²c² + b²d² - a²c² - 2abcd - b²d²
= a²d² - 2abcd + b²c²
= (ad - bc)² ≥ 0

∴ 原不等式成立
```

### 3.2 排序不等式

设 a₁ ≤ a₂ ≤ ... ≤ aₙ，b₁ ≤ b₂ ≤ ... ≤ bₙ
则顺序和 ≥ 乱序和 ≥ 逆序和

### 3.3 琴生不等式

若 f(x) 为凸函数，则
```
f((x₁ + x₂ + ... + xₙ)/n) ≤ (f(x₁) + ... + f(xₙ))/n
```

## 第四章 证明策略

### 4.1 构造函数法

将不等式转化为函数问题。

**例题**：证明 sinx < x（x > 0）

**证明**：
```
设 f(x) = sinx - x
f'(x) = cosx - 1 ≤ 0

∴ f(x) 在 (0, +∞) 上单调递减
且 f(0) = 0

∴ 当 x > 0 时，f(x) < f(0) = 0
即 sinx < x
```
