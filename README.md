# 全学段数学证明导师系统

## 📚 项目简介

全学段数学证明导师系统是一个集成初中、高中、大学三个学段的智能数学证明辅导平台。通过先进的内容识别和智能路由技术，系统能够自动识别题目所属学段，并提供针对性的分析、指导和练习推荐。

## 🎯 核心功能

### 1. 智能学段识别
- 自动识别题目属于初中、高中还是大学
- 基于关键词和语义分析的高精度分类
- 支持手动切换学段

### 2. 分学段分析
- **初中(v1.0)**：平面几何、代数证明
- **高中(v3.0)**：8大题型全覆盖
- **大学(v7.1)**：14门课程深度分析

### 3. 证明方法指导
- 各学段核心证明方法详解
- 关键步骤拆解
- 常见错误预警

### 4. 评分标准
- 中考评分标准
- 高考评分标准
- 大学严谨性标准

### 5. 个性化推荐
- 根据学习情况智能推荐练习题
- 跨学段知识图谱推荐

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Flask 2.3+
- Node.js (可选，用于前端开发)

### 安装依赖

```bash
cd backend
pip install -r requirements.txt
```

### 启动后端服务

```bash
cd backend
python app.py
```

服务将在 `http://localhost:5000` 启动。

### 访问前端

直接用浏览器打开 `frontend/index.html` 文件，或配置Web服务器。

## 📖 API文档

### 1. 学段识别
```
POST /api/identify_level
{
    "content": "证明三角形全等..."
}

Response:
{
    "level": "junior_high",
    "confidence": 0.95
}
```

### 2. 统一分析
```
POST /api/analyze
{
    "content": "用数学归纳法证明...",
    "level": "auto"
}

Response:
{
    "level": "senior_high",
    "domain": "数列与归纳法",
    "analysis": {...},
    "recommendations": [...]
}
```

### 3. 学段切换
```
POST /api/switch_level
{
    "level": "university"
}
```

### 4. 学习报告
```
GET /api/report?user_id=xxx
```

## 📝 开发指南

### 添加新的知识库内容
1. 在对应学段目录下创建 `.md` 文件
2. 更新分析器中的知识映射
3. 重启服务

### 添加新的题库
1. 在对应学段题库目录下创建 `.json` 文件
2. 遵循题库格式规范
3. 更新分析器中的题库加载逻辑

### 扩展新的学段
1. 在 `backend/` 下创建新目录
2. 实现分析器接口
3. 在 `app.py` 中注册路由
